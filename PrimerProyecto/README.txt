Pasos para usar el proyecto


1. Descomprimir el archivo
2. Abrir la carpeta con VSCode
3. Abrir una terminal
4. Usar dentro de la terminal "python manage.py runserver" para correr el programa
5. Ingresar a  http://127.0.0.1:8000/familia para ver la pagina principal
6. Para agregar miembros de la familia usar el boton AGREGAR. Esto se puede usar para llevar un registro de los familiares y sus fechas de nacimiento/cumplea√±os
Para borrar familiares una vez cargados, usar el boton BORRAR
7. Para ingresar a evento apretar el boton EVENTO
8. Dentro del evento podemos agregar familiares que asistan o no para llevar un conteo de la gente que va, para eso usar el boton AGREGAR
Para borrar familiares una vez cargados, usar el boton BORRAR
9. Volver a la http://127.0.0.1:8000/familia 
10. Para ingresar a tareas apretar el boton TAREAS
11. Dentro de la ventana tareas podemos agregar, usando el boton AGREGAR, que tarea va a realizar cada persona, en que horario y cuanto tiempo
les va a llevar (siempre en minutos el tiempo).
Para borrar familiares una vez cargados, usar el boton BORRAR
12.Volver a la http://127.0.0.1:8000/familia
